Add Your Own GeoJSON

Load custom territorial boundaries from any public GeoJSON URL. Perfect for adding your own datasets, regional borders, or specialized administrative divisions.

Requirements:

  • Must be a valid GeoJSON file
  • URL must be publicly accessible (CORS-enabled)
  • Recommended: use CDN or GitHub raw URLs
  • Max file size: ~10MB for performance
